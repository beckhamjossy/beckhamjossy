## Featured projects

### 🔧 [AWS Auto-Troubleshooter](https://github.com/beckhamjossy/aws-auto-troubleshooter)
Serverless diagnostic tool that automatically runs health checks across EC2, S3, and IAM when a CloudWatch alarm fires — delivering a structured report to Slack within seconds.
Built with Lambda · EventBridge · CloudWatch · Python · Bash
> Inspired by internal tooling I built at AWS that reduced case resolution time by **30%**

### 🛡️ [Amazon Aegis](https://github.com/beckhamjossy/amazon-aegis) *(internship project)*
Contributed to an AWS-internal monitoring and incident automation platform — built diagnostic workflows, rule-based health checks, and runbook automation across EC2, S3, IAM, and networking layers.

### 📚 [Bunge Digital Library](https://github.com/beckhamjossy/bunge-library)
Full DSpace deployment for the Parliament of Kenya — end-to-end configuration of metadata schemas, ingestion workflows, role-based access, and server-side setup (Tomcat, PostgreSQL, REST API).
> Reduced manual document handling by **30%+** through automated ingestion workflows
